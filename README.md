# Skriuw

### Verdict & Credits for ?
I've been wanting to built a desktop application for a very long time, I've tried but always quit due the horrible DX Electron has. Tried Tauri but eventually abbandoned the project because of various technical reasons: Rust being intimidating, Sync engines (PGlite, Zero, Replicache and my own), me being overly picky on designs and rich text editors being a pain to work with.

Eventually I came accros this note taking application which I lost when I reinstalled my OS. it was a Svelte application with exactly the UI how I wanted it. Seemed unmaintaned and licensed in such way that one could freely recreate the code without copying. Which I did, in Tauri 2.0, Next.JS and InstantDB. A lot of UI aspects are close to identical to that application. Down the road i'll be implementing more and more of my own sauce and I also offer a broader spectrum of features which require custom UI. So to sum up; much love for the  forgotten minimalistic notes application which lead me to finally building my own application which I scrapped dozens of times due to UI.
