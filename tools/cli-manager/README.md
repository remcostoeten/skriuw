# Skriuw CLI Manager

Interactive CLI tool for managing the Skriuw monorepo.

## Quick Start

From the project root:

```bash
bun run cli
# or
bun run sk
# or
./sk
```

## Documentation

Complete documentation is available at: **[localhost:3000/docs/cli](http://localhost:3000/docs/cli)**

Or visit the docs sections:
- [Getting Started](http://localhost:3000/docs/cli/getting-started)
- [Configuration](http://localhost:3000/docs/cli/configuration)
- [Features](http://localhost:3000/docs/cli/features)
- [Usage Guide](http://localhost:3000/docs/cli/usage)

## Development

```bash
# Run with hot reload
bun run dev

# Build for production
bun run build

# Run built version
bun run start
```

## Configuration

Edit `src/config.ts` to customize apps, commands, and settings.

## Requirements

- Bun 1.1.0+
- Node.js 18+
- Git (for repository features)
- Vercel CLI (optional, for deployment)
