### Try It Now!

1. **Create a heading:**
   ```
   Type: ##
   Press: Space
   Result: Heading 2!
   ```

2. **Make text bold:**
   ```
   Type: **bold text**
   Result: bold text (formatted instantly)
   ```

3. **Create a list:**
   ```
   Type: -
   Press: Space
   Result: вҖў List item
   ```

## 2пёҸвғЈ @ Mention Linking

### The Problem

Before: "Check the requirements document" (which one? where?)

After: "Check @Requirements Document" (click to navigate!)

### How It Works

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ Type @                          в”Ӯ
в”Ӯ                                 в”Ӯ
в”Ӯ в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ    в”Ӯ
в”Ӯ в”Ӯ Search notes...         в”Ӯ    в”Ӯ
в”Ӯ в”ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Ө    в”Ӯ
в”Ӯ в”Ӯ > Requirements Document в”Ӯ    в”Ӯ

в”Ӯ в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ    в”Ӯ
в”Ӯ                                 в”Ӯ
в”Ӯ Select and it becomes:          в”Ӯ
в”Ӯ Check @Requirements Document вҶҗ  в”Ӯ
в”Ӯ                        (click!)  в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

### Step-by-Step

1. **Type @**
   - Dropdown appears with all notes

2. **Type to filter**
   ```
   @req вҶ’ Shows "Requirements Document"
   @api вҶ’ Shows "API Integration"
   ```

3. **Select note**
   - Press `Enter` OR
   - Click with mouse

4. **Click to navigate**
   - Click the `@mention` link
   - Instantly jump to that note!

- `вҶ“` вҶ’ Next note

## рҹҶҳ Troubleshooting

### "Markdown not converting"
вҶ’ Make sure you press `Space` after shortcuts like `##`

### "Dropdown not appearing"
вҶ’ Make sure you have multiple notes created

### "Can't find my note"
вҶ’ Type to filter: `@requ` finds "Requirements"

### "Link not working"
вҶ’ Make sure you selected a note (it should be highlighted)
