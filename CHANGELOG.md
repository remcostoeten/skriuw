# Released Features

This document tracks the major features released in Skriuw.

## Core Features
- [**Command Palette**](./apps/web/content/features/command-palette.md): Centralized command execution and shortcut management.
- **Note Editor**: Rich text editing with block-based content.
- **File System**: Folder/File hierarchy with drag-and-drop.

## Administration
- [**Admin Tools**](./apps/web/content/features/admin-tools.md): User seeding and template management.
- **Dev Widget**: Real-time status and debugging tools.

## System
- **Authentication**: Secure login via Better-Auth.
- **Storage**: Postgres-backed persistent storage.
