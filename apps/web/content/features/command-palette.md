# Command Palette & Shortcuts

Power users can navigate Skriuw efficiently using the Command Palette and keyboard shortcuts.

## Command Palette
**Trigger**: `Ctrl+P` (or `Cmd+P` on Mac) / `Cmd+K`

The Command Palette allows you to:
- **Navigation**: Jump to Settings, Collections, or Toggle Sidebar.
- **File Operations**: Create new notes, folders, or save current work.
- **View Options**: Toggle Theme, Split View, or Sidebar.
- **Editing**: Undo, Redo, and other editor actions.

## Keyboard Shortcuts

### General
| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| **Command Palette** | `Ctrl+P` / `Cmd+K` | `⌘+P` / `⌘+K` |
| **New Note** | `Ctrl+N` | `⌘+N` |
| **New Folder** | `Ctrl+F` | `⌘+F` |
| **Save Note** | `Ctrl+S` | `⌘+S` |
| **Open Settings** | `Ctrl+,` | `⌘+,` |
| **Rename Item** | `Ctrl+R` | `⌘+R` |
| **Delete Item** | `Del` | `Del` |

### View & Navigation
| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| **Toggle Sidebar** | `Ctrl+B` | `⌘+B` |
| **Toggle Theme** | `Alt+T` | `Alt+T` |
| **Focus Editor** | `/` | `/` |

### Split View
| Action | Shortcut |
|--------|----------|
| **Toggle Split** | `Ctrl+\` |
| **Swap Panes** | `Shift+Ctrl+\` |
| **Force Vertical** | `Ctrl+Alt+V` |
| **Force Horizontal** | `Ctrl+Alt+H` |
| **Focus Left/Right** | `Ctrl+Alt+←/→` |
| **Close Pane** | `Ctrl+Alt+W` |

*Tip: You can remapping shortcuts in the Settings menu.*
