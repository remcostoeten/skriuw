# Shortcuts System

The shortcuts system allows users to perform actions quickly with keyboard combinations. It's event-based, database-backed, and fully customizable.

## How It Works

```
1. Shortcuts stored in InstantDB (shortcuts entity)
2. ShortcutProvider registers all enabled shortcuts on mount
3. User presses keys → system dispatches custom events
4. Components listen for events and respond
```

## Core Hooks

### useGlobalShortcuts()

Automatically registers all shortcuts from the database. Already included in `ShortcutProvider`, so you don't need to call this manually.

```typescript
// Already in app/layout.tsx via Providers
<ShortcutProvider>
  {children}
</ShortcutProvider>
```

### useComponentShortcuts(bindings)

Register custom handlers for specific shortcuts in your components.

```typescript
import { useComponentShortcuts } from '@/modules/shortcuts';

function NoteEditor() {
  const { createNote } = useCreateNote();
  const { toggleSearch } = useSearch();

  useComponentShortcuts([
    {
      id: 'new-note',           // Matches action in DB
      handler: () => createNote(),
    },
    {
      id: 'toggle-search',
      handler: () => toggleSearch(),
    },
  ]);

  return <div>...</div>;
}
```

**Options:**
```typescript
useComponentShortcuts(bindings, { enabled: true });
```

### useModifierKeys()

Track which modifier keys are currently pressed.

```typescript
const { ctrl, shift, alt, meta } = useModifierKeys();

if (ctrl && shift) {
  // Show special UI
}
```

## Event System

The shortcuts system uses custom events for loose coupling:

### Dispatched Events

| Event | Triggered By | Purpose |
|-------|-------------|---------|
| `search:toggle` | `Cmd/Ctrl+F` | Toggle search panel |
| `folders:toggle` | `Cmd/Ctrl+0` | Toggle folders sidebar |
| `note:create` | `Cmd/Ctrl+N` | Create new note |
| `folder:create` | `Cmd/Ctrl+Shift+N` | Create new folder |
| `preferences:open` | `Cmd/Ctrl+,` | Open settings |

### Listening for Events

```typescript
useEffect(() => {
  const handler = () => {
    console.log('Search toggled!');
  };
  
  window.addEventListener('search:toggle', handler);
  return () => window.removeEventListener('search:toggle', handler);
}, []);
```

| Combo | Action | Description |

| `Cmd/Ctrl+0` | `toggle-folders` | Toggle folders panel |

| `Cmd/Ctrl+Shift+N` | `new-folder` | Create new folder |

## Creating Shortcuts Programmatically

### Use Mutations

```typescript
import { useCreateShortcut } from '@/modules/shortcuts';

function SettingsPanel() {
  const { createShortcut } = useCreateShortcut();

  const addCustomShortcut = async () => {
    await createShortcut({
      action: 'custom-action',
      combo: 'Cmd/Ctrl+Shift+K',
      description: 'My custom action',
      enabled: true,
      global: true,
    });
  };

  return <button onClick={addCustomShortcut}>Add Shortcut</button>;
}
```

### Update Shortcuts

```typescript
import { useUpdateShortcut } from '@/modules/shortcuts';

const { updateShortcut } = useUpdateShortcut();

await updateShortcut({
  id: shortcutId,
  combo: 'Cmd/Ctrl+K',  // Change keybinding
  enabled: false,        // Disable it
});
```

### Delete Shortcuts

```typescript
import { useDestroyShortcut } from '@/modules/shortcuts';

const { destroyShortcut } = useDestroyShortcut();

await destroyShortcut(shortcutId);
```

## Adding New Actions

To add a new action to the system:

### 1. Add to Type Definition

```typescript
// modules/shortcuts/types.ts
export type TShortcutAction =
  | 'toggle-search'
  | 'new-note'
  | 'your-new-action';  // Add here
```

### 2. Add Handler in hooks.ts

```typescript
// modules/shortcuts/hooks.ts
function createShortcutHandler(action: TShortcutAction) {
  switch (action) {
    // ... existing cases
    case 'your-new-action':
      return () => {
        window.dispatchEvent(new CustomEvent('your:event'));
      };
    // ...
  }
}
```

### 3. Listen in Your Component

```typescript
useEffect(() => {
  const handler = () => {
    // Do something
  };
  
  window.addEventListener('your:event', handler);
  return () => window.removeEventListener('your:event', handler);
}, []);
```

### 4. Create in Database

```typescript
await createShortcut({
  action: 'your-new-action',
  combo: 'Cmd/Ctrl+Y',
  description: 'Your action description',
  enabled: true,
  global: true,
});
```

## Combo Format

Shortcuts use a string format with modifiers and keys:

```typescript
'Cmd/Ctrl+F'           // Cmd on Mac, Ctrl on Windows/Linux
'CmdOrCtrl+Shift+N'    // Multiple modifiers
'Alt+K'                // Alt key
'Shift+Backspace'      // Shift with special key
'Escape'               // Single key, no modifiers
```

**Platform Handling:**
- `CmdOrCtrl` automatically becomes `Cmd` on Mac, `Ctrl` elsewhere
- Matching is case-insensitive
- Special keys: `Backspace`, `Enter`, `Escape`, `Tab`, `Space`, `ArrowUp`, etc.

## Prevention Logic

Shortcuts are prevented in certain contexts:

```typescript
// modules/shortcuts/utils.ts
export function shouldPreventShortcut(event: KeyboardEvent): boolean {
  const target = event.target as HTMLElement;
  const tagName = target.tagName.toLowerCase();

  // Don't trigger in inputs, textareas, or contenteditable
  if (tagName === 'input' || tagName === 'textarea') return true;
  if (target.isContentEditable) return true;

  return false;
}
```

This prevents shortcuts from firing while typing in the note editor or search box.

## Schema

Shortcuts are stored in InstantDB:

```typescript
shortcuts: i.entity({
  action: i.string(),           // Unique action identifier
  combo: i.string(),            // Key combination
  description: i.string(),      // Human-readable description
  enabled: i.boolean(),         // Can be disabled
  global: i.boolean(),          // System-wide vs app-only
  createdAt: i.number().indexed(),
  updatedAt: i.number(),
})
```

## API Reference

### Queries

- `useGetShortcuts()` - All shortcuts
- `useGetEnabledShortcuts()` - Only enabled shortcuts

### Mutations

- `useCreateShortcut()` - Create new shortcut
- `useUpdateShortcut()` - Update existing shortcut
- `useDestroyShortcut()` - Delete shortcut

### Utilities

- `matchesShortcut(event, combo)` - Check if KeyboardEvent matches combo string
- `shouldPreventShortcut(event)` - Check if shortcut should be blocked
- `setupDefaultShortcuts(existing)` - Initialize default shortcuts

## Example: Full Implementation

```typescript
import { useComponentShortcuts, useCreateShortcut } from '@/modules/shortcuts';

function MyFeature() {
  const [showPanel, setShowPanel] = useState(false);
  const { createShortcut } = useCreateShortcut();

  // Register component-specific shortcuts
  useComponentShortcuts([
    {
      id: 'toggle-my-panel',
      handler: () => setShowPanel(prev => !prev),
    },
  ]);

  // Create shortcut in DB on mount
  useEffect(() => {
    createShortcut({
      action: 'toggle-my-panel',
      combo: 'Cmd/Ctrl+Shift+P',
      description: 'Toggle my custom panel',
      enabled: true,
      global: true,
    }).catch(console.error);
  }, []);

  return (
    <div>
      {showPanel && <Panel />}
    </div>
  );
}
```
