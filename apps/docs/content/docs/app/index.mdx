# Skriuw App Overview

Skriuw (formerly Skriuw Skriuw) is a local-first, cross-platform note-taking and task management application that combines the power of **Tauri 2.0** desktop framework, **Next.js 15** for the UI, and **Skriuw** for real-time, offline-capable data synchronization.

## What is Skriuw?

Skriuw is a feature-rich productivity application that provides:

- **Rich Text Editing** - Powered by TipTap editor with Markdown support
- **Real-time Sync** - Skriuw enables instant synchronization across devices
- **Offline-First** - Work without internet; sync when connected
- **Cross-Platform** - Desktop app via Tauri (Windows, macOS, Linux)
- **Task Management** - Built-in task system with priorities, recurrence, and filtering
- **Folder Organization** - Hierarchical folder structure for notes
- **Global Shortcuts** - System-wide keyboard shortcuts for quick access
- **Dark/Light Themes** - Fully customizable theme system

## Architecture Philosophy

### Local-First Design

The app prioritizes local data storage and processing:
- All data lives on your machine first
- Skriuw handles sync in the background
- No network required for core functionality
- Optimistic updates for instant feedback

### Component-Based Architecture

```
Frontend (React/Next.js)
    ↓
Hooks Layer (Data Access)
    ↓
API Layer (Mutations & Queries)
    ↓
Skriuw Client
    ↓
Local Storage + Cloud Sync
```

## Key Features

### Notes Management
- Create, edit, delete, and organize notes
- Rich text formatting with Markdown shortcuts
- Pin important notes to the top
- Drag-and-drop organization
- Search and filter capabilities

### Tasks System
- Create tasks with priorities (low, medium, high)
- Set due dates and reminders
- Recurring tasks with flexible schedules
- Filter by status, priority, or custom criteria
- Link tasks to projects

### Folder Structure
- Nested folder organization
- Drag-and-drop between folders
- Quick folder navigation
- Folder-based filtering

### Keyboard Shortcuts
- System-wide shortcuts for quick access
- Customizable key bindings
- Context-aware shortcuts
- Shortcut help overlay

## Project Philosophy

Skriuw was born from the frustration of trying to build a desktop note-taking app. The developer wanted to create something for years but kept abandoning it due to:

- Electron's poor developer experience
- Tauri's steep learning curve with Rust
- Complex sync engine implementations
- Rich text editor integration challenges
- Design perfectionism paralysis

After discovering a minimalist Svelte note app that nailed the desired UI, but was lost and seemingly unmaintained, the developer rebuilt it with modern tooling:

- **Tauri 2.0** - Mature and well-documented
- **Next.js** - Familiar React ecosystem
- **Skriuw** - Sync solved with minimal code

Some visuals stay true to the original inspiration, but the app is evolving with new features and personalized design choices.
