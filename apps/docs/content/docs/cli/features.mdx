# Features

The CLI Manager is packed with features to streamline your development workflow.

## Interactive Menu System

### ASCII Art Logo
- Beautiful startup screen with customizable ASCII art
- Professional, minimalist aesthetic
- No emojis (clean, focused interface)

### Arrow-Key Navigation
- Intuitive menu navigation using ↑/↓ keys
- Quick selection with Enter key
- Organized into logical sections:
  - Development
  - Build
  - Deploy
  - Utilities

### Dynamic Menu Options
- Automatically generated from configuration
- Shows app names, ports, and status
- Context-aware menu items
- Easy to customize and extend

## Development Features

### Run Individual Apps

Start any configured app with a single selection:

```
❯ Run Tauri App (port 42069)
  Run Docs App (port 3000)
```

Each app shows:
- Display name
- Port number
- Running status
- Path
- Local URL

### Run All Apps

Start all configured apps simultaneously:
- Launches apps sequentially with delays
- Shows progress for each startup
- Returns to menu when all are running
- Non-blocking terminal

### Process Management

Advanced process handling:
- **Non-blocking execution** - Terminal stays responsive
- **Background processes** - Apps run independently
- **Output filtering** - Suppresses noisy framework logs
- **Clean status displays** - Structured info boxes
- **Auto-detection** - Knows when apps are ready

### Status Display

Beautiful status boxes for running apps:

```
┌────────────────────────────────────────┐
│  Tauri App                             │
├────────────────────────────────────────┤
│  Status:      Running                  │
│  Port:        42069                    │
│  Path:        ./apps/instantdb         │
│  URL:         http://localhost:42069   │
└────────────────────────────────────────┘
```

## Hotkey Controls

Quick actions with single keypresses:

### O - Open in Browser
- Opens `http://localhost:[port]` in default browser
- Works for any running app
- Instant feedback

### C - Open in Code Editor
- Opens app directory in your configured editor
- Default: Cursor
- Configurable for any editor (VS Code, Vim, etc.)

### R - Restart App
- Stops the app
- Waits for clean shutdown
- Starts it again
- Shows progress

### S - Stop App
- Gracefully terminates the app process
- Removes from running apps list
- Returns to menu

### I - Install Package
- Interactive package name prompt
- Runs `bun add [package]` in app directory
- **Does not stop the dev server**
- Shows installation progress

### Multi-App Selection

When multiple apps are running:
1. Shows list of all active processes
2. Arrow-key navigation to select
3. Apply hotkey to selected app

```
? Select an app to manage:
❯ ● Tauri App (port 42069)
  ● Docs App (port 3000)
  ← Back
```

## Build & Deployment

### Build Individual Apps

Build a single app with detailed output:
- Shows build progress
- Displays duration
- Success/failure status
- Path information

### Build All Apps

Batch build with comprehensive summary:

```
┌────────────────────────────────────────┐
│  Build Summary                         │
├────────────────────────────────────────┤
│  Total Apps:    2                      │
│  Successful:    2                      │
│  Failed:        0                      │
│  Duration:      45.32s                 │
└────────────────────────────────────────┘
```

### Deploy to Staging

Deploy to Vercel preview environment:
- Runs `vercel deploy`
- Shows deployment progress
- Displays deployment URL
- Returns to menu when complete

### Deploy to Production

Deploy to Vercel production:
- Runs `vercel deploy --prod`
- Requires confirmation (via Vercel CLI)
- Shows deployment status
- Production URL displayed

## Repository Management

### Open Repository

Automatically opens your GitHub repository:
1. Runs `git remote -v`
2. Parses the remote URL
3. Opens in default browser
4. Works with GitHub, GitLab, Bitbucket, etc.
