# Skriuw Documentation

Welcome to the Skriuw documentation! Skriuw is a local-first, near-instant sync note-taking application built with Tauri 2.0, Next.js, and Skriuw.

## What is Skriuw?

Skriuw is a modern note-taking application that combines the best of desktop and web experiences:

- **Local-First Architecture** - Your data lives on your device first
- **Near-Instant Sync** - Changes sync across devices seamlessly
- **Cross-Platform** - Desktop app (Tauri) and web app (Next.js)
- **Offline Support** - Works without an internet connection
- **Real-Time Collaboration** - Powered by Skriuw

## Project Structure

The Skriuw monorepo consists of:

### Applications

- **Tauri App** (`apps/web`) - Desktop application built with Tauri 2.0
- **Web App** (`apps/web`) - Next.js web application
- **Documentation** (`apps/docs`) - This documentation site (Fumadocs)

### Tools

- **Seeder** (`tools/seeder`) - Database seeding utilities
- **Servo** (`tools/servo`) - Development launcher for building and running applications

## Quick Start

### Using Servo (Recommended)

The easiest way to get started is to use the Servo development launcher:

```bash
# Build and run Servo
cd tools/servo
./build.sh
./servo
```

This launches an interactive menu where you can:
- Run applications (dev server)
- Build for production
- Deploy to Vercel
- Run tools like the seeder

### Manual Setup

```bash
# Install dependencies
bun install

# Run the main app
cd apps/web
bun run dev

# Run the docs site
cd apps/docs
bun run dev
```

## Tech Stack

### Core Technologies

- **Tauri 2.0** - Desktop application framework
- **Next.js 15** - React framework with TypeScript
- **Skriuw** - Real-time database with offline support
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **TipTap** - Rich text editor

### Development Tools

- **Bun** - Fast JavaScript runtime and package manager
- **Fumadocs** - Documentation framework
- **Vercel** - Deployment platform
